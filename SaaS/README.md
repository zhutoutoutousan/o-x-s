# Lovers SaaS Platform

A low-code drag-and-drop platform for creating beautiful lover's sites.

## Structure

- `packages/builder` - Drag-and-drop builder engine
- `packages/timeline` - Timeline component
- `packages/chat` - Chat/messaging component
- `packages/effects` - Visual effects (candles, fire)
- `packages/video` - Video player component
- `packages/legacy` - Legacy museum for elderly users
- `packages/app` - Main SaaS application

## Getting Started

```bash
pnpm install
pnpm dev
```
